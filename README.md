# 🎬 The Ultimate Pro AI Video Agent (F5-TTS Voice Cloning 🇹🇭)

แอปพลิเคชันระดับโปรที่รันแบบ **100% Local / Offline บน Google Colab** (ไม่พึ่งพา API ภายนอก) ซึ่งถูกอัปเกรดมาให้รองรับระบบ **Voice Cloning (โคลนเสียง) ภาษาไทยแบบสมจริง** และ **สามารถวิเคราะห์วิดีโอความยาว 1 ชั่วโมงเต็มได้โดยที่ระบบไม่ค้าง!**

## ✨ ฟีเจอร์ใหม่และจุดเด่นล่าสุด (The Ultimate Update):
1. 🧠 **Long-Video Chunking Analysis:** หมดปัญหาวิดีโอยาวแล้ว AI อ่านไม่จบ! ระบบใหม่จะทำการซอยบทสนทนา (Chunking) เป็นท่อนย่อยๆ แล้วให้ AI สกัดหาไฮไลต์จากต้นจนจบคลิป (1 ชั่วโมงก็อ่านหมด) 
2. 📊 **Real-time Progress Bar:** ไม่ต้องลุ้นว่าระบบค้างไหม! หน้าเว็บมีหลอดโหลดบอกสถานะ (Progress Bar) เปอร์เซ็นต์ความคืบหน้าให้เห็นแบบเรียลไทม์ตั้งแต่ตอนวิเคราะห์ จนถึงตอนเรนเดอร์ภาพและเสียง
3. 🎨 **Advanced Subtitles & Audio:** เพิ่มหน้าต่างการตั้งค่าขั้นสูง (Advanced Settings) ให้คุณปรับแต่งสีซับไตเติ้ล (เหลือง, ขาว, เขียว) ขนาดตัวอักษร และระดับความดังของเพลงประกอบ (BGM) ได้ตามใจชอบ
4. 🧹 **Auto Garbage Collection:** ระบบเคลียร์ไฟล์ขยะชั่วคราวทิ้งโดยอัตโนมัติ (แม้จะเกิด Error) ทำให้เครื่อง Colab ของคุณพื้นที่จะไม่เต็ม
5. 🗣️ **Auto Voice Cloning (F5-TTS-TH-V2):** ระบบหาท่อนเสียงพูดที่ดีที่สุดไปโคลนให้อัตโนมัติ (และมีช่องให้อัปโหลดเสียงพากย์เองได้)
6. 🙎‍♂️ **Smart Auto-Crop:** ครอปภาพแนวตั้ง (9:16) แบบติดตามใบหน้า (Face Tracking) ด้วย `MediaPipe` คนไม่หลุดเฟรมแน่นอน
7. 🗂️ **Video Gallery & Management:** หน้าแดชบอร์ดสำหรับเรียกดู เล่น ดาวน์โหลด หรือลบวิดีโอที่คุณสร้างไว้ในระบบ

## ⚙️ เทคโนโลยีที่ใช้เบื้องหลัง (ทำงานบน GPU ของ Colab)
*   **ถอดข้อความเสียง & ซับไตเติ้ล:** `faster-whisper`
*   **AI วิเคราะห์และสรุปสคริปต์:** `Qwen/Qwen2.5-1.5B-Instruct`
*   **สร้างเสียงพากย์และโคลนเสียงภาษาไทย:** `f5-tts-th` (โมเดล **VIZINTZOR/F5-TTS-TH-V2**)
*   **วิเคราะห์ติดตามใบหน้า:** `MediaPipe` (Google)
*   **ประมวลผลวิดีโอ/เสียง:** `FFmpeg`

## 🚀 วิธีการนำไปใช้งานบน Google Colab

### วิธีที่ 1: อัปโหลดไฟล์ Notebook ไปเปิดบน Colab (ง่ายที่สุด)
1. ดาวน์โหลดไฟล์ `Chotta_Local_Colab.ipynb` จากโปรเจกต์นี้
2. เปิด [Google Colab](https://colab.research.google.com/) เลือกเมนู **"Upload"** และอัปโหลดไฟล์เข้าไป
3. ตรวจสอบให้แน่ใจว่าเปิดใช้ GPU แล้ว (เมนู `Runtime` -> `Change runtime type` -> เลือก `T4 GPU`)
4. กดรันเซลล์ทั้งหมดตามลำดับ (คลิกที่เซลล์แล้วกด `Shift + Enter`)
   *   *(หมายเหตุ: เซลล์แรกๆ จะทำการติดตั้งไลบรารีแบบล็อคเวอร์ชันตาม `requirements.txt` และดาวน์โหลดโมเดลมาเตรียมไว้ให้ก่อน)*

### วิธีที่ 2: รันโค้ดและติดตั้งผ่านไฟล์ requirements.txt
หากคุณต้องการตั้งค่าด้วยตัวเอง ให้รันสคริปต์อัตโนมัติ `setup.py` ที่เตรียมไว้ให้:
```bash
python setup.py
python app.py
```
